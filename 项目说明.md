## 项目目标

构建一个基于 LangChain + RAG 的本地 PDF 知识库问答系统，
支持上传 PDF 文档并进行智能问答，同时展示引用来源。

## 关键交付物

- [x] Streamlit Web 交互界面
- [x] PDF 上传与解析
- [x] 文本自动切块与向量化
- [x] FAISS 向量库检索
- [x] DeepSeek LLM 问答
- [x] 引用来源展示
- [x] 多轮对话支持
- [x] 向量库持久化
- [x] 环境配置检查
- [ ] 多文档联合推理
- [ ] 文档摘要功能

## 约束条件

- Embedding 模型本地运行（CPU）
- LLM 通过 API 调用（需网络）
- 单次最大文档数：不限（取决于内存）
