## 1. 项目概述

基于 LangChain + RAG 的本地 PDF 知识库问答系统。用户上传 PDF 文件后，
系统自动切片、向量化构建索引，用户可对文档内容进行智能问答，
并查看引用来源。

## 3. 核心流程

### 3.1 PDF 索引流程

1. 用户上传 PDF（支持批量）
2. 临时文件写入 → PyPDFLoader 加载
3. RecursiveCharacterTextSplitter 切块（chunk_size=500, overlap=100）
4. BGE-small-zh-v1.5 Embedding 模型向量化
5. FAISS 向量库建立索引
6. 索引持久化到 outputs/vectorstore/
7. 清理临时文件

### 3.2 问答流程

1. 用户输入问题
2. FAISS similarity_search 检索 Top-3 相关文档块
3. 构建带上下文的 Prompt
4. DeepSeek LLM 生成回答（temperature=0.3）
5. 返回回答 + 来源引用

## 5. 优化说明 (v2.0)

### 5.1 代码质量
- **去重**：统一 process_pdf() 函数，消除 main() 中重复逻辑
- **错误处理**：PDF 加载、LLM 调用、向量检索均增加 try/except
- **环境检测**：启动时检查 DEEPSEEK_API_KEY 配置

### 5.2 功能增强
- **向量库持久化**：索引完成后自动保存，下次启动自动加载
- **清空索引**：新增一键清空功能
- **进度反馈**：批量处理时显示进度条
- **文件校验**：仅接受 .pdf 格式文件

### 5.3 配置管理
- llm_config.py：API Key 缺失时抛出明确错误
- .env.example：提供完整的配置模板和说明
- max_tokens 可通过环境变量 LLM_MAX_TOKENS 配置
